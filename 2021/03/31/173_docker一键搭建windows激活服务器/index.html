docker一键搭建windows激活服务器

现在自己安装Windows系统都需要激活，如果想在百度里找激活码基本上都无法激活Windows 10，网上下载的激活工具又不敢用，那么为什么不自己建一个KMS服务器呢？通过自己建的KMS服务器直接激活自己的Windows 10是不是很方便呢？

《Docker 安装 KMS 服务器 激活Windows 10》

今天我来使用vlmcsd ，这是一款和 Microsoft 兼容的 KMS 服务器，是 KMS 服务器的一个开源实现，接下来就用 docker 来搭建一个KMS服务器。

腾迅云安装Parrot系统

使用腾迅云安装Parrot系统十分方便. 大致流程如下

  1. 【腾讯云】1核2G5M轻量应用服务器50元首年，高性价比，助您轻松上云
    https://cloud.tencent.com/act/cps/redirect?redirect=1556&cps_key=2e3bc7971ddd1eedd18322c07bedb646&from=console
  2. 下载. 从 https://parrotsec.org/virtual/ 下载回来 OVA文件 并解压 得到 vmdk文件
  3. 上传到腾迅云COS 。浏览文件 点击 复制临时链接
  4. 导入镜象 https://console.cloud.tencent.com/cvm/image/index?rid=5 选强制导入
  5. 修改引导 完成

Parrot使用文档: https://www.parrotsec.org/docs/
